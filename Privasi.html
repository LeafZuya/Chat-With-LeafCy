Kebijakan Privasi

Diciptakan Pada:19 Mei 2026
Terakhir diperbarui: 30 Juni 2026 [Menambahkan Easter Egg Rahasia di LeafCy & Leafia, Cari Sendiri 😹]

1. Informasi yang Kami Kumpulkan

LeafCy AI dirancang dengan menghormati privasi pengguna. Data yang disimpan hanya terbatas pada:

  • Riwayat Percakapan - Tersimpan secara lokal di perangkat Anda (LocalStorage browser) untuk memberikan pengalaman percakapan yang berkelanjutan.
  • Preferensi Pengguna - Seperti mode gelap/terang, status BGM, dan progres tantangan.
  • Batas Penggunaan - Hitungan pesan per jam untuk menjaga kualitas layanan.
Penyimpanan Lokal 100% — Tidak ada data yang dikirim ke server penyimpanan eksternal. Semua percakapan Anda hanya ada di browser Anda sendiri.

2. Pemrosesan AI & API Pihak Ketiga

Untuk memproses pesan dan menghasilkan respons, LeafCy AI menggunakan proxy API yang mengakses model AI dari pihak ketiga (OpenRouter/Server Zyrion Leaf-Ice). Berikut yang perlu Anda ketahui:

  • Pesan Anda dikirim ke API AI hanya pada saat Anda mengirim pesan untuk mendapatkan respons.
  • API tidak menyimpan percakapan Anda secara permanen — setiap permintaan bersifat stateless.
  • Kami menggunakan Workers Proxy untuk melindungi identitas dan privasi Anda.
Catatan Penting: Jangan pernah membagikan informasi pribadi sensitif (password, nomor KTP, data perbankan, dll) dalam percakapan dengan AI mana pun, termasuk LeafCy.

3. Hak Hapus Data (True Memory Delete)

LeafCy AI memiliki fitur "True Memory" yang memberikan kontrol penuh atas data Anda:

  • Hapus satu pesan — Klik ikon tong sampah di hover pesan → Pesan dihapus permanen dan AI akan langsung melupakannya.
  • Hapus seluruh riwayat chat — Semua pesan dalam satu percakapan dihapus total, AI kehilangan semua memori dari chat tersebut.
  • Chat Baru — Membuat percakapan baru tanpa menghapus yang lama.
Fitur Hapus = Hapus Ingatan AI — Setiap kali Anda menghapus pesan atau chat, AI akan benar-benar melupakan percakapan yang dihapus tersebut. Ini adalah komitmen privasi maksimal.

4. Batasan Penggunaan (Rate Limit)

Untuk menjaga kestabilan layanan dan mencegah penyalahgunaan, LeafCy menerapkan batasan:

  • Maksimal 20 pesan teks per jam per pengguna.
  • Waktu reset otomatis setiap 90 menit.
  • Fitur tantangan "Zuya Ganteng 😎" dapat mengurangi waktu reset hingga 30 menit.

Batasan ini hanya berlaku untuk jumlah pesan, bukan untuk menyimpan atau melacak identitas pribadi Anda.

5. Penggunaan Penyimpanan Lokal (LocalStorage)

LeafCy AI menggunakan teknologi LocalStorage browser untuk menyimpan:

  • Seluruh riwayat percakapan Anda
  • Preferensi mode gelap/terang
  • Status pemutaran BGM
  • Progres tantangan
  • Data batas penggunaan

Data ini tidak pernah meninggalkan perangkat Anda. Menghapus cache/history browser atau menggunakan mode incognito akan menghapus semua data LeafCy AI Anda.

6. Privasi untuk Anak-Anak

LeafCy AI dirancang dengan karakter yang ramah anak (usia 7+ tahun). Namun:

  • Orang tua disarankan untuk mendampingi anak saat menggunakan layanan AI.
  • Jangan membagikan informasi pribadi seperti alamat rumah, nomor telepon, atau data sekolah.
  • Kami tidak dengan sengaja mengumpulkan data dari anak di bawah 7 tahun tanpa persetujuan orang tua.

7. Keamanan Data

Langkah-langkah keamanan yang kami terapkan:

  • Proxy server untuk mencegah paparan langsung ke API AI.
  • Validasi input dasar untuk mencegah injeksi berbahaya.
  • Semua data tetap di perangkat Anda — tanpa server penyimpanan pusat.

Namun perlu diingat: Tidak ada sistem yang 100% aman. Gunakan layanan AI dengan bijak.

8. Perubahan Kebijakan Privasi

Kami dapat memperbarui kebijakan privasi ini dari waktu ke waktu. Perubahan signifikan akan diberitahukan melalui:

  • Pemberitahuan di aplikasi LeafCy AI
  • Update tanggal "Terakhir diperbarui" di bagian atas halaman ini

Dengan terus menggunakan LeafCy AI setelah perubahan, Anda dianggap menyetujui kebijakan privasi yang telah diperbarui.
